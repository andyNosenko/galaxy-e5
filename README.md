## Важные замечания

Перед началом процесса убедитесь, что на вашем ноутбуке достаточно свободного места для создания резервной копии. Рекомендуется иметь не менее 20-30 гигабайт свободного пространства.

## Шаги для создания резервной копии

1. **Запуск ADB теста**:
    - Сначала запустите скрипт `adb_test.bat`. Этот скрипт проверит корректность настроек ADB и подключение вашего устройства.

2. **Создание ZIP-резервной копии**:
    - Если тест завершился успешно, запустите скрипт `backupzip.bat`.
    - Обратите внимание, что создание ZIP-резервной копии может занять около 5 минут.

3. **Создание полной резервной копии**:
    - После завершения операции с ZIP, запустите скрипт `backup.bat`.
    - Этот процесс может занять больше времени — около 15-20 минут.
   
4. **Восстановление всех папок на ГУ после создания полной резервной копии**:
    - После завершения операции с Создание полной резервной копии, запустите скрипт `restorezip.bat`.
    - Этот процесс может занять больше времени — около 15-20 минут.

## Важные советы

- После завершения этих операций вы можете попробовать переводить необходимые приложения и загружать их локализации на русском и английском языках.
- **Важно**: Не пытайтесь вносить изменения или производить перепрошивку устройства, если резервная копия не была успешно создана. Это может привести к потере данных.

## Заключение

Следуйте этим шагам внимательно, чтобы обеспечить успешное создание резервной копии вашего устройства. Помните о важности наличия свободного пространства и успешного завершения всех операций.
